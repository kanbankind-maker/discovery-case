# Сегмент B: Семья (30–45)

## Профиль
- Возраст: 30–45 лет
- Доля аудитории: 31% (крупнейший сегмент)
- Средний чек: 3 500–8 000 ₽
- Промозависимость: 68%

## Воронка (данные за 6 мес.)

| Этап | Конверсия |
|---|---|
| Banner→Entry | 29% |
| Entry→Browse | 68% |
| Browse→Cart | 21% |
| Cart→Checkout | 58% |
| Checkout→Purchase | 39% |

## Главный дроп-офф

**Browse→Cart (21%)** — пользователи просматривают товары, но не добавляют в корзину. Интервью и CJM указывают на одну причину: недоверие к продавцу. Отсутствие верифицированных отзывов, неясные условия возврата и незнакомые бренды создают ощущение риска при покупке на значительную для семейного бюджета сумму.

## Контекст из данных

- `data/funnel.csv` — Browse→Cart у сегмента B = 21%, при этом у пользователей, открывших карточку продавца, конверсия в корзину вырастает до 34% (+62%).
- `interviews/segment_b.md` — 8 из 12 респондентов упомянули «не знаю этого продавца» как причину остановки; 5 из 12 ждут, пока кто-то из знакомых купит первым.
- `cjm/segment_b.md` — момент наивысшего сомнения зафиксирован на странице товара при прокрутке к блоку «О продавце»: пользователи ищут рейтинг, срок работы, политику возврата.
- `tickets/legal_review.md` — персонализация на основе банковских данных (история транзакций, категории трат) одобрена юридическим отделом. Это позволяет показывать «Этот продавец популярен среди покупателей с похожим профилем» без раскрытия PII.

## Открытые вопросы для дискавери

1. Какие именно сигналы доверия работают для этого сегмента — рейтинг продавца, число продаж, срок на платформе или знакомый бренд?
2. Готовы ли пользователи 30–45 разрешить использование банковской истории для персональных рекомендаций в обмен на снижение риска покупки?
3. Насколько велика разница в доверии между «известным брендом на маркетплейсе» и «неизвестным продавцом с высоким рейтингом» — есть ли порог рейтинга, достаточный для конверсии?

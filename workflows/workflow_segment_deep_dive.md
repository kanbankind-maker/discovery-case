# Workflow: Segment Deep Dive

**Цель:** Полный анализ одного сегмента — от данных до гипотез  
**Время:** ~45–60 мин  
**Сегмент:** [A / B / C / D / E] — выбери один перед стартом

---

## Шаги

### Шаг 1 — Анализ воронки
- Используй: `.cursor/agents/agent_funnel_analyst.md`
- Подставь нужный сегмент в промпт
- ✅ Результат сохранён в `output/` → переходи к шагу 2

### Шаг 2 — Синтез интервью
- Используй: `.cursor/agents/agent_interview_synthesizer.md`
- ✅ Результат сохранён в `output/` → переходи к шагу 3

### Шаг 3 — Генерация гипотез
- Используй: `.cursor/agents/agent_hypothesis_generator.md`
- Агент читает output шагов 1 и 2 — убедись, что файлы есть
- ✅ Результат сохранён в `output/` → переходи к шагу 4

### Шаг 4 — Challenge-сессия (ручной шаг)
Для каждой гипотезы задай вопрос:
> «При каком условии этот вывод неверен?»

Запиши ответы прямо в файл гипотез (добавь раздел `## Challenge`).

### Шаг 5 — Опциональный питч
- Если нужен питч → `.cursor/agents/agent_pitch_writer.md`

# Agent: CJM Builder

**Роль:** UX Researcher  
**Вход:** `cjm/segment_[X].md` + `interviews/segment_[X].md`  
**Выход:** `output/YYYY-MM-DD_cjm-updated-[segment].md`

---

## Промпт

```
Ты — UX Researcher. Открой `cjm/segment_[СЕГМЕНТ].md` и `interviews/segment_[СЕГМЕНТ].md`.

Задача: дополни CJM новыми болями и инсайтами из интервью.

Для каждого этапа CJM:
1. Сохрани существующие данные (не удаляй).
2. Добавь новые боли из интервью — с цитатой и источником (file:line).
3. Отметь этапы без данных интервью — явно пометь «данных нет».
4. Выдели топ-1 самую острую боль на всём пути — обоснуй почему.

Формат: обновлённая таблица CJM + раздел «Что добавлено» + «Топ-1 боль».

Не изменяй исходные данные CJM, только добавляй.
Сохрани в `output/YYYY-MM-DD_cjm-updated-[segment].md`.
```
